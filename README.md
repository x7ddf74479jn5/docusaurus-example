## 構成

- Node.js：18.7.0
- React：17.0.2
- @docusaurus/core: 2.0.1
- @docusaurus/preset-classic：2.0.0-beta.15
- @mdx-js/react: ^1.6.22

## 環境立ち上げ

ライブラリインストール

```shell
yarn install
```

サーバ起動

```shell
yarn start
```

ブラウザでアクセス

```shell
localhost:3000
```
