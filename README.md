## 構成

- Node.js：16.13.0
- React：17.0.1
- Docusaurus：2.0.0-beta.15
- @docusaurus/preset-classic：2.0.0-beta.15
  - Infima：0.2.0-alpha.36

Docusaurus 2 系は開発中のバージョンであるため、バージョンアップ対応などは、より注意して行うこと。

## 環境立ち上げ

ライブラリインストール

```shell
yarn install
```

サーバ起動

```shell
yarn start
```

ブラウザでアクセス

```shell
localhost:3000
```
